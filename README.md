# AG Stevens Game Dev

[![Build Status](https://travis-ci.org/SakuyaIzayoi/agsgd.svg?branch=master)](https://travis-ci.org/SakuyaIzayoi/agsgd)

## Build Instructions
1. `mkdir build && cd build`
2. `cmake .. && make`
3. Execute by running `./Agsgd`

## Testing
1. `mkdir build && cd build`
2. `cmake .. && make`
3. Execute by running `./agsgd_test`
